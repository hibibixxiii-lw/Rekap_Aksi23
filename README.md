# Rekap Aksi – Aplikasi Rekap Keuangan Paroki

Aplikasi desktop sederhana untuk merekap aksi seperti:
- Aksi Natal
- APP
- APG
- Aksi lainnya

Dirancang ringan, portable, dan mudah digunakan untuk pelayanan.

---

## Fitur Utama

- Input cepat (Speed Entry)
- Format Rupiah otomatis
- Running total otomatis
- Edit data (Double Click)
- Undo data terakhir
- Export ke Excel & PDF
- Auto Backup setiap 5 menit
- Backup otomatis saat aplikasi ditutup
- Limit 30 backup terakhir

---

## Cara Install

1. Install Python 3.10 atau lebih baru
2. Install dependency:

pip install -r requirements.txt

---

## Cara Menjalankan

python main.py
